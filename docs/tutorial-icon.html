Tutorial: icon

icon

Icon Component

A lightweight Vue component that dynamically loads and renders SVG icons from the local assets folder.

It uses Vite’s import.meta.glob to import all SVG files as raw strings and renders the selected icon via v-html.

Purpose

This component allows you to:

  • Dynamically render SVG icons by name
  • Avoid importing each icon manually
  • Keep icon usage consistent across the project

Props

name

Type Required Description
String Yes The filename of the icon (without .svg)

Example

If you pass:

<Icon name="calendar" />
